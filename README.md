# HTTP HEADERS | Fridge Combo #111

A hyper-minimalist HTTP header generator for developers. Created for those moments when you need a perfectly formatted header string for cURL or API testing without opening heavy tools.

## Core Philosophy
- **1 Tool / 1 Action**: Just generate the header and copy it.
- **Developer First**: Focus on the headers you use most (Bearer, JSON, API Key).
- **Zero Friction**: No login, no project setup, just instant results.

## Features
- **Instant Toggle**: Switch between `cURL` format (-H "...") and `Raw` format (Key: Value) with one tap.
- **Smart Presets**: Quick chips for common headers like Authorization and Content-Type.
- **Copy-Centric UI**: Tap the output area or the main button to instantly copy to your clipboard.
- **iOS Optimized**: Native-feel UI with long-press menu prevention for a clean "tool" experience.

## Usage
1. Select the header type using the top chips.
2. Paste your token, key, or custom value into the input field.
3. Choose your output format (cURL or Raw).
4. Tap the result area or "COPY HEADER" button.
